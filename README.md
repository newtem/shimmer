# shimmer

## Overview
Shimmer is a lightweight interpreter designed for simple debugging and testing of the Net Dust 2.2 language
It provides a streamlined environment to execute Net Dust 2.2 code instantly without complex build processes

## Goals
- Enable real-time interpretation and debugging of Net Dust 2.2 scripts

## Environment
- OS: Windows (WinForm)
- Language: C# (.NET)
- Key Libraries:
**[Fody](https://github.com/Fody/Fody)**: Used via NuGet to embed DLL dependencies into the primary EXE, enabling a single-file deployment
- Google Gemini: refine the internal logic of `netdust.cs`
